# Mod Cores das Pedras

Modificação do arquivo `ItemList`(da versão BR) para melhorar a legibilidade dos nicks... trocando as cores difíceis de se enxergar definidas pelas pedras.  

*Novas versões são geradas automaticamente pelo GitHub Actions sempre que o jogo atualiza o arquivo(a verificação é feita diariamente por volta das 15h).*

## Como usar

1. Baixe a versão mais recente [aqui](https://github.com/0Z1O/Pedras/releases/latest).
2. Extraia/substitua o arquivo na pasta do jogo.
3. Inicie o jogo normalmente.

## Aviso

Utilize por sua conta e risco!  
Recomenda-se fazer um backup do arquivo antes de substituí-lo.
